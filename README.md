## 📄 **프로젝트 개요서**

### 📍 **프로젝트 개요**
- **프로젝트 명**: 하우스웨이
- **개발 주제**: 대학생들을 위한 원룸 찾기 서비스. 매물을 올린 공인중개사와 사용자를 연결해 주는 서비스.
- **개발 기간**: 3주
- **사용 기술**:
    - JDK 17.0.13
    - Spring Boot 2.7.18
    - MySQL
    - MyBatis
    - MariaDB
    - Git Hub
    - Intellij IDEA
    - Tomcat version 8.5.63
 

### 🎯 **프로젝트 목표**
- 사용자가 매물을 조회하고 예약할 수 있는 **부동산 백엔드 시스템** 구축
- **회원 가입** 및 **로그인** 기능 구현, API 인증 적용
- 관리자가 매물을 관리할 수 있는 관리용 API 제공
- RESTful API 설계 및 개발
- Spring Boot 기반의 백엔드 구조 설계 및 구현
- MariaDB와 MyBatis를 통한 데이터베이스 연동
- 단위 테스트 및 통합 테스트 작성으로 코드 품질 향상
